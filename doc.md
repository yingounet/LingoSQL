## 📋 项目简介

LingoSQL 是一个现代化的 Web 数据库管理工具，支持 MySQL 和 PostgreSQL 数据库的管理。采用 Golang 后端 + Vue 前端的架构，提供直观的用户界面和强大的数据库管理功能。

**核心特性：**
- 多数据库支持（MySQL、PostgreSQL）
- 用户认证与授权系统
- 数据库、表、用户的快速创建与管理
- SQL 查询执行与结果展示
- 查询收藏功能
- 查询历史记录
- 未来 AI 功能：自然语言生成 SQL、SQL 错误检查

## 🚀 快速开始

### 技术栈概览

**后端：**
- Golang + Gin
- SQLite（服务端数据库）
- MySQL/PostgreSQL 驱动

**前端：**
- Vue 3 + TypeScript
- Element Plus
- Monaco Editor

**部署：**
- Docker 单容器部署
- 单端口（8080）

### 开发环境

```bash
# 后端
cd backend
go run cmd/server/main.go

# 前端
cd frontend
npm install
npm run dev
```

### 生产部署

```bash
# 构建镜像
docker build -t lingosql:latest -f backend/Dockerfile .

# 运行容器
docker run -d \
  --name lingosql \
  -p 8080:8080 \
  -v $(pwd)/data:/data \
  -e DB_PATH=/data/lingosql.db \
  -e JWT_SECRET=your-secret-key \
  lingosql:latest
```
